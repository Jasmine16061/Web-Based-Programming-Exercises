# Web-Based Programming Course - Exercises

This repository contains all the weekly exercises, practical assignments, during the Web-Based Programming semester course. It tracks the complete learning journey from foundational markup languages to advanced backend server integration.

---

## Learning Roadmap & Core Modules

The repository is organized according to the course's weekly hands-on training milestones:

### 1. HTML5 Fundamentals (Structure & Semantics)
* Core tags, document structures, text formatting, and lists (`2`, `3`, `4`).
* Deep dive into Block vs. Inline elements and layout architecture (`5`).
* Intermediate components: Tables, Media components, and Form handling (`6`, `7`, `8`, `9`).
* Internal and External hyperlinking strategies, favicons, and metadata tags (`10`, `11`, `12`).

### 2. Styling & Modern Layouts
* Core styling rules, selectors, and cascading stylesheets (`13. CSS`).

### 3. Client-Side Programming (JavaScript ES6+)
* Introduction to programming logic, control structures, and variable definitions (`14`).
* Object-Oriented programming concepts and state modeling in JavaScript (`15`).
* Comprehensive frontend logic dynamic integration hub (`18. JavaScript Learning Hub`).

### 4. Advanced Backend & Software Engineering Tools
* Introduction to Java-based server-side architecture (`16. Java Servlet`).
* Implementation of web data inputs and relational customer data storage (`17. Eclipse Scientific Calculator & MySQL integration`).

---

## Environment & Tools Used
* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Backend & Database:** Java Servlet, MySQL
* **Development IDEs:** VS Code, Eclipse
* **Version Control:** Git & GitHub Management
